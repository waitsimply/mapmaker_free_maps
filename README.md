# Free Maps for Teachers and Journalists

A comprehensive collection of high-quality, free map images designed for educational and journalistic use. All maps are created using [mapmaker.io](https://mapmaker.io), a professional map-making platform.

## About This Repository

This repository provides free access to a curated selection of maps suitable for:

- **Teachers and Educators** - Use in classrooms, presentations, worksheets, and educational materials
- **Journalists and Media** - Illustrate news stories, reports, and articles
- **Students and Researchers** - Support academic projects and presentations
- **Non-profit Organizations** - Create informative materials and reports

## How to Use These Maps

1. Browse the `/maps/` directory to find the map you need
2. Download the high-resolution version
3. Use freely in your educational or journalistic materials
4. Attribution to mapmaker.io is required

## Map Formats

Maps are available in multiple formats:
- **PNG** - High-resolution images for print and digital use
- **SVG** - Vector graphics for scalability and editing
